Nous sommes engagés à rendre notre site web accessible à tous les utilisateurs, y compris ceux ayant des limitations physiques ou cognitives. Nous nous efforçons de respecter les normes d'accessibilité WCAG 2.1 de niveau AA. Nous avons pris plusieurs mesures pour améliorer l'accessibilité de notre site web, notamment :

  • Utilisation de couleurs à contraste élevé pour faciliter la lisibilité
  • Utilisation de balises d'en-tête pour organiser le contenu de manière logique
  • Utilisation de descriptions alternatives pour les images afin que les utilisateurs malvoyants puissent comprendre leur contenu
  • Fourniture d'options de zoom pour faciliter la lecture du contenu pour les utilisateurs ayant une vision réduite
  • Utilisation de polices sans-serif pour améliorer la lisibilité
  • Utilisation de la sémantique HTML appropriée pour faciliter la compréhension du contenu par les utilisateurs ayant des lecteurs d'écran.
Nous continuerons à travailler pour améliorer l'accessibilité de notre site web et à nous conformer aux normes d'accessibilité en vigueur. Si vous rencontrez des difficultés d'accessibilité ou si vous avez des suggestions pour améliorer l'accessibilité de notre site web, n'hésitez pas à nous contacter.
